    一、基础概念

    想深入了解matplotlib，首先需要了解两个概念：

    • figure:画图板/画布，所有的操作未明确指定画布则默认在最近的figure上进行操作
    • axes：图形区/绘图区，用来绘图的实际区域，一个画布上可以创建一个或多个绘图区。可以直接设定pyplot.axes([x, y, w, h])，更一般的方式是通过subplot来获取绘图区

    二、画布figure

    创建画布

    plt.figure创建

    • num指定画布的标号，不指定的话plt会默认增量式的新建一个画布
    • figsize指定画布的大小, 但是还没创建axes
    import matplotlib.pyplot as plt
fig = plt.figure(num=1, figsize=(5,5))

    plt.subplots同时创建figure和axes

    fig, axes = plt.subplots(1,2) # 创建了axes维度
axes[0].plot([1,2,3,4])
axes[0].set_title('test1')
axes[1].plot([4,3,2,1])
axes[1].set_title('test2')

    隐式的创建figure

    如果之前未创建figure, 直接运行plt.plot或者 plt.subplot会先隐式的创建一个figure.

    ax1 = plt.subplot(111) #为当前figure创建一个axes，默认为plt.subplot(111)
ax1.plot([1,2,3,4])

plt.plot([1,2,3,4]) #也可以直接plt.plot,相当于上面两句话

    在画布上新增axes

    # 方式一
fig.add_subplot(122)

# 方式二
plt.subplot(122)

    获取当前画布或axes

    # 获取当前figure，若不存在，创建一个新figure
fig = plt.gcf() 

# 获取当前axes
plt.gca() 

# 根据axes获取画布
fig = ax.figure

    保存画布

    fig.savefig('a.jpg', dpi=100, bbox_inches='tight', progressive=True)

    清除画布或axes

    
 # 清除axes，即当前 figure 中的活动的axes，但其他axes保持不变  
plt.cla()  

# 清除当前 figure 的所有axes，但是不关闭这个 window，所以能继续复用于其他的 plot
plt.clf()     

# 关闭 window，如果没有指定，则指当前 window  
plt.close()
